# Final Homework: Trino + PostgreSQL + MySQL

## Описание

Интеграция через Trino и последующая визуализация/сохранение данных в Iceberg.

### Основные этапы:
1. **Подключение через Trino** к PostgreSQL и MySQL
2. **Отдельно для PostgreSQL и MySQL**:
   - **Агрегация данных**
   - **Визуализация**: Построение нескольких графиков по агрегированным данным
   - **Iceberg**: Создание схемы и таблицы, сохранение агрегированных данных, проверка содержимого.

#### Данные
1. Для **PostgreSQL** был взят образ ghusta/postgres-world-db:latest - содержит датасет с данными о странах, городах, языках, населении, континентах, экономике
2. Для **MySQL** был взят образ sakiladb/mysql:latest - содержит датасет с таблицами фильмов, актеров, арендаторов, клиентов, платежей, магазинов, сотрудников
