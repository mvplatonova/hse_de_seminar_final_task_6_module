# Final Homework: Trino + PostgreSQL + MySQL

## Описание

Интеграция через Trino и последующая визуализация/сохранение данных в Iceberg.

### Основные этапы:
1. **Подключение через Trino** к двум источникам: PostgreSQL и MySQL
2. **Агрегация данных**: Выполнение запросов, получение агрегированных метрик по странам и континентам
3. **Визуализация**: Построение нескольких графиков по агрегированным данным
4. **Iceberg**: Создание схемы и таблицы, сохранение агрегированных данных, проверка содержимого.
